Converts a list of Hashes into a big string that can be send to the database. The individual parts components of each entry will be separated by a null character. The entries will be separated by a new line. The first entry will be the header that you give this function.

Parameters
dataThe hashes you want to put in the string.
Parameters
sizeThis is basically a second return value. The size of the string that is returned will be put in here. The initial value that you give it will be ignored.
Parameters
headerThe header that you want in front of the string. Even if the header is empty, it will still put an end line in the beginning.
Parameters
authorsAuthor data that the spider returns.
prevCommitTimeTime of the commit before the version we parsed now.
unchangedFilesThe files we did not change from the previous version.
Returns
A char pointer to string with all the data in it.

Will match the inputlist rawData with inputlist hashes to give where they overlap. The output of this function is both the size that the author data will take up in when send to the database and the actual strings that we are going to send.

Parameters
authorsThe output map. Will give for each Hash in hashes a list of strings which contain who worked on the given function.
hashesThe function we want to find the corresponding authors for.
rawDataThe raw author data which we are going to match with the hashes.
Returns
The size of all the author strings.
