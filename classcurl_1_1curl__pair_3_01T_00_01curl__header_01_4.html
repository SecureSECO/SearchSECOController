Detailed Description

template<class T>
class curl::curl_pair< T, curl_header >

Template specialization for curl_header type. Why do we need this? Because curl_header wraps a struct curl_slist list of headers. libcurl functions can't handle a curl_header type, so we need to specialize curl_pair to return a struct curl_slist *.
